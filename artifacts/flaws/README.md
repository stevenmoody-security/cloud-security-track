# flAWS Artifacts

Write-ups for flAWS levels.

## Naming
`YYYY-MM-DD_flaws_level-XX.md`

## Standard
Each artifact should answer:
- what failed (misconfig/control gap)
- blast radius
- attacker path (high-level)
- guardrail (prevent at scale)
- detection (signals + one detection idea)
- evidence link (redacted)
